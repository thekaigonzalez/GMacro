GMacro is a markup language written in D, for D.

GMacro is open source and you can download the source code 
on the releases page.

GMacro is licensed under the GNU GPL 3.0 License.

You can view the source code on GitHub.

Examples:

PROJECT(hello-world, no-dep);

How to Build:

    You can build using the .\build command on windows, or using make on Unix devices.
